# 六爻金钱卦

一个基于三枚铜钱六次起卦的六爻金钱卦网页应用。用户输入所问之事后，网站会引导自下而上完成六次起卦，并生成本卦、变卦、动爻、卦辞和分项解析。

## 功能

- 三枚铜钱六次起卦
- 本卦、变卦和动爻展示
- 64 卦卦名、卦辞、象意和白话参考
- 事业、财运、家庭、姻缘、行动建议分项解读
- 乾隆通宝风格铜钱界面
- 复制结果摘要和重新起卦
- 桌面与移动端响应式布局

## 本地运行

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
```
